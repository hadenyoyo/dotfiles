# Haden's Dotfiles

- Clone repo
```
sudo pacman -S stow git
```

`git clone https://github.com/hadenyoyo/dotfiles/`

- `cd dotfiles`, `stow .`

## Required packages
- `hyprland`
- `stow`
- `hyprpaper`
- `waybar`
- `hyprshot`
- `fastfetch`
- `rofi-wayland`
- `firefox`
- `nwg-display`
- `pipewire`, `pipewire-pulse`, `wireplumber`
- `pavucontrol`

```
sudo pacman -S hyprland stow hyprpaper waybar fastfetch rofi-wayland firefox ttf-hack otf-font-awesome nwg-display pipewire pipewire-pulse wireplumber pavucontrol
yay -S hyprshot
```

### Fonts
- `ttf-hack`
- `otf-font-awesome`
- `ttf-jetbrains-mono`
- Windows fonts (optional)

### Optional
- `hyprpicker`
- `spotify-launcher`
