# Haden's Dotfiles

- Clone repo  
`git clone https://github.com/hadenyoyo/dotfiles/`

- `cd dotfiles`, `stow .`

## Required packages
- `hyprland`
- `stow`
- `hyprpaper`
- `waybar`
- `hyprshot`

```
sudo pacman -S hyprland stow hyprpaper waybar hyprshot
```

### Optional
- `hyprpicker`
- `spotify-launcher`

Bluetooth
- `bluez`, `bluez-utils`, `blueman`

```
sudo pacman -s hyprpicker spotify-launcher bluez bluez-utils blueman
```
